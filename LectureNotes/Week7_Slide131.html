t-SNE Algorithm

t-Distributed Stochastic Neighbor Embedding

🌐 Non-linear Dimensionality Reduction
Preserves local structure while revealing global patterns
📊 Probability Distributions
Models similarities as conditional probabilities
C = KL(P||Q) = ΣΣ p_ij log(p_ij/q_ij)
🔔 Heavy-Tailed Distribution
Uses Student t-distribution to avoid crowding problem

Algorithm Steps

  • Compute High-D Similarities
    Calculate pairwise similarities in original space
  • Random Initialization
    Initialize low-dimensional points randomly
  • Calculate Q Distribution
    Compute similarities in embedded space
  • Optimize Positions
    Minimize KL divergence via gradient descent
  • Early Exaggeration Phase
    Multiply P by factor to form tight clusters
