Bootstrap Aggregating (Bagging)

🎒 What is Bagging?

Bootstrap Aggregating (Bagging) trains multiple models on different random samples of the training data, then combines their predictions through voting or averaging.

🎯 Key Insight

"Wisdom of Crowds" - Many weak learners, each with different perspectives, can together make better decisions than any single strong learner.

📊 Bootstrap Sampling

Sample with replacement from original dataset. Each bootstrap sample has same size but different composition.

🔄 Bagging Process

  • 1
    Bootstrap Sample
    Create random sample with replacement from training data (typically same size as original)
  • 2
    Train Model
    Train a base learner (e.g., decision tree) on this bootstrap sample
  • 3
    Repeat
    Repeat steps 1-2 for N iterations (typically 100-1000 models)
  • 4
    Aggregate
    Combine predictions: majority vote (classification) or average (regression)
