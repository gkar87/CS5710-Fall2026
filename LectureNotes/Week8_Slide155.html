Optimizers: SGD, Adam, RMSprop

SGD with Momentum

v_t = βv_{t-1} + (1-β)∇f θ_t = θ_{t-1} - αv_t β: momentum (0.9) α: learning rate

Pros:

  • Simple and reliable
  • Good generalization
  • Low memory usage

Cons:

  • Requires LR tuning
  • Slower convergence

Adam

m_t = β₁m_{t-1} + (1-β₁)∇f v_t = β₂v_{t-1} + (1-β₂)∇f² m̂_t = m_t/(1-β₁ᵗ) v̂_t = v_t/(1-β₂ᵗ) θ_t = θ_{t-1} - α·m̂_t/(√v̂_t+ε)

Pros:

  • Adaptive learning rates
  • Fast convergence
  • Works well out-of-box

Cons:

  • Can overfit
  • Higher memory usage

RMSprop

v_t = βv_{t-1} + (1-β)∇f² θ_t = θ_{t-1} - α·∇f/√(v_t+ε) β: decay rate (0.9) α: learning rate ε: small constant (1e-8)

Pros:

  • Adapts per-parameter
  • Good for RNNs
  • Handles sparse gradients

Cons:

  • No bias correction
  • Can get stuck

Optimizer Comparison

Optimizer Best For Learning Rate Memory Speed
SGD Computer Vision 0.01-0.1 Low Slow
SGD+Momentum CNNs, General 0.001-0.01 Low Medium
Adam Most tasks, NLP 0.0001-0.001 2x params Fast
RMSprop RNNs, Online 0.001-0.01 1x params Fast
AdaGrad Sparse data 0.01 1x params Fast
AdamW Transformers 0.0001-0.001 2x params Fast
