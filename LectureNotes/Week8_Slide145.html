Activation Functions (Sigmoid, Tanh, ReLU)

Sigmoid

σ(x) = 1 / (1 + e^(-x))
  • Range: (0, 1)
  • Smooth gradient
  • Vanishing gradient problem

Tanh

tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))
  • Range: (-1, 1)
  • Zero-centered
  • Stronger gradients than sigmoid

ReLU

ReLU(x) = max(0, x)
  • Range: [0, ∞)
  • No vanishing gradient
  • Dead neuron problem

When to Use Each Activation Function

Function Best For Advantages Disadvantages
Sigmoid Output layer (binary classification) Probability interpretation Vanishing gradient
Tanh Hidden layers in RNNs Zero-centered output Still has vanishing gradient
ReLU Hidden layers in deep networks Fast computation, no saturation Dead neurons, not zero-centered
