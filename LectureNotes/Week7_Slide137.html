Wrapper Methods

What are Wrapper Methods?

Wrapper Methods evaluate feature subsets by training and testing machine learning models. They "wrap" around a specific algorithm to find the optimal feature combination for that algorithm.

Common Algorithms

🔄 Recursive Feature Elimination (RFE)
Recursively removes features and builds model on remaining attributes
⏩ Forward Selection
Starts with no features, iteratively adds the best feature
⏪ Backward Elimination
Starts with all features, iteratively removes worst feature
🔀 Bidirectional Search
Combines forward and backward search for optimal results

Wrapper vs Filter vs Embedded

🎯
Filter Methods
Independent of ML algorithm
  • Very fast
  • Algorithm agnostic
  • Ignores feature interactions
  • No model feedback
🔄
Wrapper Methods
Uses specific ML algorithm
  • Considers feature interactions
  • Algorithm-specific optimization
  • Computationally expensive
  • Risk of overfitting
🧬
Embedded Methods
Built into ML algorithm
  • Automatically selects features
  • Good balance of speed/accuracy
  • Algorithm-specific
  • Limited control over process
