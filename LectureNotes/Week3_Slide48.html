Multi-class Classification (One-vs-All)

One-vs-All (OvA)

One-vs-All converts a K-class problem into K binary problems. For each class k, train a classifier to distinguish "class k" from "all other classes".
Train K classifiers: h_θ^(k)(x)
One binary classifier per class
Prediction: Choose class with highest probability

OvA vs OvO

🎯
One-vs-All
  • K classifiers
  • 🎯 Simpler training
  • 📈 Scales well
  • ✅ Binary classifiers
⚡
One-vs-One
  • K(K-1)/2 classifiers
  • 📉 Less data per classifier
  • 📈 Good for large K
  • ✅ Binary classifiers

The Algorithm

  • 1
    For class k: create binary labels
    y_i^k = 1 if y_i = k else 0
  • 2
    Train logistic regression on y^k
    h_θ^(k)(x) = P(y=k|x; θ)
  • 3
    Repeat for k = 1 to K
    Train K classifiers
  • 4
    At test time: pick class with max probability
    y_pred = argmax_k h_θ^(k)(x)
