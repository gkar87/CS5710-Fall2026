Multi-class Classification (One-vs-All)

🎯 One-vs-All Strategy

One-vs-All (OvA) extends binary classifiers to handle multiple classes by training one binary classifier for each class against all other classes.
For K classes:

Classifier i: P(y = i | x) vs P(y ≠ i | x)

Decision: class = argmax P(y = i | x)
                i ∈ {1,2,...,K}
Key Characteristics:

• Trains K binary classifiers
• Each classifier: "Class i" vs "Not Class i"
• Prediction: highest confidence score
• Works with any binary classifier

⚙️ Training Process

  • Step 1: Data Transformation
    For each class i, create binary labels: yi = 1, others = 0
  • Step 2: Train Classifiers
    Train K binary logistic regression models independently
  • Step 3: Prediction
    For new input x, compute Pi(x) = σ(θiTx) for all i
  • Step 4: Decision
    Choose class with maximum probability: argmax Pi(x)
Advantage: Can use any binary classifier (SVM, Neural Network, etc.) and probabilities don't need to sum to 1.

Each classifier outputs the probability that the input belongs to its positive class.
