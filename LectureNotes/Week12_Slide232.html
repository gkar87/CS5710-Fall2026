SHAP Values

Shapley Values Theory

Core SHAP Formula
φᵢ(f, x) = Σ (|S|!(n-|S|-1)! / n!) [f(S ∪ {i}) - f(S)]
Where φᵢ is the Shapley value for feature i, representing its contribution to the prediction difference from the average.
SHAP Properties
Efficiency: Sum of all SHAP values equals prediction difference
Symmetry: Equal features get equal attributions
Dummy: Irrelevant features get zero attribution
Additivity: Values compose across models
Intuitive Understanding
Think of SHAP as answering: "How much does each player contribute to the team's success?" It considers all possible coalitions and fairly distributes credit.
