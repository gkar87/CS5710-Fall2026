    Data Collection

  • 2
    Exploratory Data Analysis
    Understand data structure, distributions, and patterns
  • 3
    Data Cleaning
    Handle missing values, duplicates, and inconsistencies

    Data Transformation
    Scale, normalize, and encode features appropriately

    Data Splitting
    Divide into training, validation, and test sets
