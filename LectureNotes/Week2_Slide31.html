Polynomial Regression

Beyond Linear Relationships

Polynomial Regression extends linear regression to capture non-linear relationships by adding polynomial terms of the input features.
  • 📊 Non-Linear Patterns
    Captures curved relationships in data
  • 🔢 Feature Engineering
    Creates new features from existing ones
  • ⚡ Still Linear
    Linear in parameters, non-linear in features
  • ⚠️ Overfitting Risk
    Higher degrees can lead to overfitting

Mathematical Foundation

General Polynomial Form:
y = β₀ + β₁x + β₂x² + β₃x³ + ... + βₙxⁿ
Where:
• n = degree of polynomial
• β₀, β₁, ..., βₙ = coefficients to learn
• Each x^i creates a new feature
• Same optimization as linear regression
💡 Key Insight:
We transform the problem to make it linear in the coefficients!

Common Polynomial Degrees

1
Linear
y = β₀ + β₁x
2
Quadratic
y = β₀ + β₁x + β₂x²
3
Cubic
y = β₀ + β₁x + β₂x² + β₃x³
