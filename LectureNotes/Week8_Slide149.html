Gradient Descent in Neural Networks

Gradient Descent Variants

Batch Gradient Descent
Uses entire dataset to compute gradients
Stochastic Gradient Descent (SGD)
Updates weights using one sample at a time
Mini-Batch Gradient Descent
Balances between batch and stochastic
Momentum-Based Methods
Accelerates convergence using velocity

Key Parameters

  • 📊
    Learning Rate (α)
    Controls step size in parameter space
  • 📦
    Batch Size
    Number of samples per gradient update
  • 🔄
    Epochs
    Complete passes through dataset
  • ⚡
    Momentum (β)
    Fraction of previous update to keep
