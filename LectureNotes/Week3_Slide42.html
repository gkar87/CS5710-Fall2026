Binary vs Multi-class Classification

Binary Classification

  • ✓ Two Classes: Yes/No, True/False, 0/1
  • 📊 Output: Single probability P(class=1)
  • 🎯 Decision: Threshold-based (e.g., > 0.5)
  • 📈 Loss: Binary cross-entropy

Multi-class Classification

  • 🔢 N Classes: Multiple categories (N ≥ 3)
  • 📊 Output: Probability distribution over N classes
  • 🎯 Decision: argmax(probabilities)
  • 📈 Loss: Categorical cross-entropy

Key Differences

Aspect Binary Classification Multi-class Classification
Number of Classes Exactly 2 3 or more
Output Layer 1 neuron with sigmoid N neurons with softmax
Probability Constraint P(1) = 1 - P(0) ∑P(i) = 1 for all classes
Common Algorithms Logistic Regression, SVM Softmax Regression, One-vs-All
Evaluation Metrics ROC-AUC, Precision, Recall Confusion Matrix, Macro/Micro F1
Example Applications Spam detection, Fraud detection Image recognition, Language detection
