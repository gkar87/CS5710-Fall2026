Expectation-Maximization (EM) Algorithm

🎯 Algorithm Overview

The Expectation-Maximization (EM) algorithm is a general iterative method for finding maximum likelihood estimates in models with latent (hidden) variables. It alternates between two steps: the E-step (expectation) and the M-step (maximization).
E
Expectation Step
Calculate the expected value of the log-likelihood function with respect to the latent variables, given the current parameter estimates and observed data.
Q(θ|θ⁽ᵗ⁾) = E[log p(X,Z|θ) | X,θ⁽ᵗ⁾]
γ(zₙₖ) = p(zₙₖ=1|xₙ,θ⁽ᵗ⁾)
= πₖ⁽ᵗ⁾N(xₙ|μₖ⁽ᵗ⁾,Σₖ⁽ᵗ⁾) / Σⱼπⱼ⁽ᵗ⁾N(xₙ|μⱼ⁽ᵗ⁾,Σⱼ⁽ᵗ⁾)
  • ✓ Compute posterior probabilities
  • ✓ Calculate responsibilities
  • ✓ Fixed parameters from previous iteration
M
Maximization Step
Maximize the Q-function with respect to the parameters to obtain updated parameter estimates for the next iteration.
θ⁽ᵗ⁺¹⁾ = arg max Q(θ|θ⁽ᵗ⁾)
μₖ⁽ᵗ⁺¹⁾ = Σₙγ(zₙₖ)xₙ / Σₙγ(zₙₖ)
Σₖ⁽ᵗ⁺¹⁾ = Σₙγ(zₙₖ)(xₙ-μₖ⁽ᵗ⁺¹⁾)(xₙ-μₖ⁽ᵗ⁺¹⁾)ᵀ / Σₙγ(zₙₖ)
πₖ⁽ᵗ⁺¹⁾ = Σₙγ(zₙₖ) / N
  • ✓ Update means and covariances
  • ✓ Update mixing coefficients
  • ✓ Use responsibilities from E-step

📈 Convergence Properties

Monotonic Increase
The EM algorithm guarantees that the likelihood increases (or stays constant) at each iteration: L(θ⁽ᵗ⁺¹⁾) ≥ L(θ⁽ᵗ⁾). This ensures we're moving towards a (local) optimum.
Convergence Criteria
Stop when the change in log-likelihood falls below a threshold, or when parameter changes become negligible. Typical threshold: |L⁽ᵗ⁺¹⁾ - L⁽ᵗ⁾| < ε.
Local Optima
EM is guaranteed to converge to a local maximum of the likelihood function. Different initializations may lead to different local optima. Multiple random starts are recommended.
Convergence Rate
EM has linear convergence rate. Near the optimum, it can be slow to converge. The rate depends on the amount of "missing information" in the model.
