Batch Normalization

Core Concept

Normalize Inputs
Standardize inputs to each layer during training
Internal Covariate Shift
Reduces changes in input distribution
Learnable Parameters
Scale (γ) and shift (β) to preserve expressiveness
Training vs Inference
Different behavior during training and testing

BatchNorm Algorithm

Forward Pass:

μ_B = (1/m) Σ x_i
σ²_B = (1/m) Σ (x_i - μ_B)²
x̂_i = (x_i - μ_B) / √(σ²_B + ε)
y_i = γ x̂_i + β
During Inference:

Use running averages:
x̂ = (x - μ_running) / √(σ²_running + ε)
y = γ x̂ + β
