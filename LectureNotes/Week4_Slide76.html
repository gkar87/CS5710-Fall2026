SVM for Multi-class Classification

The Multi-class Challenge

SVMs are naturally binary classifiers designed to separate two classes. For problems with more than 2 classes, we need special strategies.
⚠️ Key Challenge: How do we extend binary SVMs to handle K > 2 classes?

Common Scenarios:

  • 🔤 Text Classification: Politics, Sports, Tech, Health
  • 🖼️ Image Recognition: Cat, Dog, Bird, Fish
  • 🏥 Medical Diagnosis: Healthy, Flu, Cold, COVID
  • 📊 Quality Control: Excellent, Good, Fair, Poor

Main Approaches

🥊 One-vs-All (OvA)
Train K binary classifiers: each class vs. all other classes combined.
K classes → K binary SVMs
Class i: {class i} vs {all others}
⚔️ One-vs-One (OvO)
Train a binary classifier for every pair of classes.
K classes → C(K,2) = K(K-1)/2 SVMs
Each pair: {class i} vs {class j}
🔧 Direct Multi-class
Modify SVM optimization to handle all classes simultaneously.
Single optimization problem
More complex but theoretically elegant

✅ One-vs-All Advantages
  • Only K classifiers needed
  • Fast training and prediction
  • Memory efficient
  • Easy to implement
  • Well-calibrated probabilities
❌ One-vs-All Disadvantages
  • Imbalanced training sets
  • Conflicting regions possible
  • May not find optimal boundaries
  • Sensitive to class imbalance
✅ One-vs-One Advantages
  • Balanced binary problems
  • More robust to outliers
  • Better boundaries per pair
  • Less sensitive to class imbalance
❌ One-vs-One Disadvantages
  • O(K²) classifiers needed
  • Slower training and prediction
  • Higher memory usage
  • Vote aggregation needed

Implementation Examples

Using Scikit-Learn (Automatic)

# Scikit-learn automatically handles multi-class from sklearn.svm import SVC from sklearn.datasets import load_iris from sklearn.model_selection import train_test_split # Load multi-class dataset (3 classes) X, y = load_iris(return_X_y=True) X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3) # One-vs-All (default for most kernels) svm_ova = SVC(kernel='rbf', decision_function_shape='ovr') svm_ova.fit(X_train, y_train) # One-vs-One svm_ovo = SVC(kernel='rbf', decision_function_shape='ovo') svm_ovo.fit(X_train, y_train) # Prediction and evaluation pred_ova = svm_ova.predict(X_test) pred_ovo = svm_ovo.predict(X_test) print(f"OvA Accuracy: {svm_ova.score(X_test, y_test):.3f}") print(f"OvO Accuracy: {svm_ovo.score(X_test, y_test):.3f}")

Manual One-vs-All Implementation

# Manual One-vs-All implementation from sklearn.svm import SVC import numpy as np class OneVsAllSVM: def __init__(self, **svm_params): self.svm_params = svm_params self.classifiers = {} self.classes = None def fit(self, X, y): self.classes = np.unique(y) # Train one classifier for each class for cls in self.classes: # Create binary labels: current class vs all others binary_y = (y == cls).astype(int) # Train binary SVM clf = SVC(**self.svm_params) clf.fit(X, binary_y) self.classifiers[cls] = clf def predict(self, X): # Get decision scores from all classifiers scores = {} for cls, clf in self.classifiers.items(): scores[cls] = clf.decision_function(X) # Predict class with highest score predictions = [] for i in range(len(X)): best_class = max(self.classes, key=lambda cls: scores[cls][i]) predictions.append(best_class) return np.array(predictions)

Performance Comparison

# Comprehensive comparison of approaches import time from sklearn.metrics import classification_report def compare_multiclass_strategies(X_train, X_test, y_train, y_test): strategies = { 'One-vs-All': SVC(kernel='rbf', decision_function_shape='ovr'), 'One-vs-One': SVC(kernel='rbf', decision_function_shape='ovo') } results = {} for name, clf in strategies.items(): # Training time start_time = time.time() clf.fit(X_train, y_train) train_time = time.time() - start_time # Prediction time start_time = time.time() predictions = clf.predict(X_test) pred_time = time.time() - start_time # Accuracy accuracy = clf.score(X_test, y_test) results[name] = { 'accuracy': accuracy, 'train_time': train_time, 'pred_time': pred_time, 'n_support_vectors': clf.n_support_, 'n_classifiers': len(clf.classes_) if name == 'OvA' else len(clf.classes_) * (len(clf.classes_) - 1) // 2 } print(f"{name}:") print(f" Accuracy: {accuracy:.3f}") print(f" Train time: {train_time:.3f}s") print(f" Pred time: {pred_time:.4f}s") print(f" # Classifiers: {results[name]['n_classifiers']}") print() return results
