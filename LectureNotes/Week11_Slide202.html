Agent-Environment Interaction

🤖
Agent
Action (aₜ)
State (sₜ₊₁), Reward (rₜ)
🌍
Environment

🤖 Agent's Perspective

  • Observes current state sₜ
  • Selects action aₜ based on policy π
  • Receives reward signal rₜ
  • Updates internal model/policy
  • Learns from experience

🌍 Environment's Role

  • Receives agent's action aₜ
  • Transitions to new state sₜ₊₁
  • Computes reward rₜ
  • May be deterministic or stochastic
  • Defines problem dynamics
