Naive Bayes Classifier

What is Naive Bayes?

Naive Bayes is a probabilistic classifier based on Bayes' theorem. It predicts the class with the highest probability given the input features.
Key Characteristics:

• Probabilistic: Outputs class probabilities, not just labels
• Simple & Fast: Linear time complexity for training and prediction
• Effective: Often works surprisingly well despite its simplicity
• Robust: Handles missing values and noise well

Formula:
P(class|features) = P(features|class) × P(class) / P(features)

The "Naive" Assumptions

  • 🔀 Feature Independence
    Features are conditionally independent given the class. This is usually unrealistic but often works well in practice.
  • 📊 Feature Distribution
    Features follow a specific distribution (usually Gaussian for continuous features).
  • 📈 Sufficient Data
    Each class has enough training examples to estimate probabilities accurately.
  • ⚖️ Class Balance
    Works best when classes are reasonably balanced, though can handle some imbalance.

How Naive Bayes Works

1
Training Phase
Calculate prior probabilities P(class) and likelihood P(feature|class) for each feature-class pair from training data.
2
Prediction Phase
For new instance, multiply prior by all feature likelihoods using the independence assumption.
3
Classification
Choose the class with the highest posterior probability. Normalization is optional since we only need relative probabilities.
