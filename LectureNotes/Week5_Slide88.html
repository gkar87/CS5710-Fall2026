XGBoost Regularization

Regularization Types

XGBoost uses both L1 and L2 regularization to control model complexity and prevent overfitting.
Regularization benefits:

• Simplicity: Favors simpler models
• Generalization: Better performance on unseen data
• Feature Selection: L1 can zero out features
• Stability: Reduces variance in predictions

Key Parameters

  • alpha (reg_alpha)
    L1 regularization on leaf weights [default=0]
  • lambda (reg_lambda)
    L2 regularization on leaf weights [default=1]
  • gamma (min_split_loss)
    Minimum loss reduction for split [default=0]
  • max_depth
    Maximum tree depth [default=6]
  • min_child_weight
    Minimum sum of instance weight in child [default=1]

Regularized Objective Function

Obj = L(y, ŷ) + Ω(f)
where Ω(f) = γT + ½λ∑w² + α∑|w|
Loss Function
L(y, ŷ)
Measures prediction error (MSE, log-loss, etc.)
Tree Complexity
γT
Penalty on number of leaves (T)
L2 Regularization
½λ∑w²
Ridge penalty on leaf weights
L1 Regularization
α∑|w|
