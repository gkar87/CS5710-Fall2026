Common Kernels

📏
Linear Kernel
K(x, y) = xTy + c
  • Complexity: O(n)
  • Parameters: c (optional constant)
  • Decision Boundary: Linear (hyperplane)
  • Overfitting Risk: Low
🎯 Best For:
• High-dimensional data (text, genes)
• Linearly separable problems
• Large datasets (fast training)
• When interpretability is crucial
📈
Polynomial Kernel
K(x, y) = (xTy + c)d
  • Complexity: O(nd)
  • Parameters: d (degree), c (constant)
  • Decision Boundary: Polynomial curves
  • Overfitting Risk: Medium-High
🎯 Best For:
• Image classification
• Natural language processing
• Problems with smooth boundaries
• When global structure matters
🎯
RBF Kernel
K(x, y) = exp(-γ||x - y||2)
  • Complexity: O(n)
  • Parameters: γ (gamma)
  • Decision Boundary: Complex, localized
  • Overfitting Risk: Medium (tunable)
🎯 Best For:
• Most general-purpose problems
• Complex non-linear patterns
• Medium-sized datasets
• When local patterns matter
