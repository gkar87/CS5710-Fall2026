Long Short-Term Memory (LSTM)

What is LSTM?

Long Short-Term Memory networks are a special type of RNN designed to solve the vanishing gradient problem and learn long-range dependencies in sequences.

Key Innovation: LSTM introduces a cell state that acts as a "memory highway" allowing information to flow unchanged across many time steps.

LSTM Cell Structure
Cell State (C_t) - Memory Highway
f_t
i_t
o_t
Hidden State (h_t) - Output

LSTM vs Vanilla RNN

Vanilla RNN Problems:

  • Single hidden state carries all information
  • Gradients vanish through many time steps
  • Cannot learn long-range dependencies
  • Information gets overwritten at each step

LSTM Solutions:

  • Dual state system (cell + hidden)
  • Gating mechanisms control information flow
  • Linear cell state enables gradient flow
  • Selective memory updating and forgetting
LSTM Equations
Forget Gate
f_t = σ(W_f · [h_{t-1}, x_t] + b_f)
Input Gate
i_t = σ(W_i · [h_{t-1}, x_t] + b_i)
C̃_t = tanh(W_C · [h_{t-1}, x_t] + b_C)
Cell State Update
C_t = f_t * C_{t-1} + i_t * C̃_t
Output Gate
o_t = σ(W_o · [h_{t-1}, x_t] + b_o)
h_t = o_t * tanh(C_t)

LSTM Advantages

🧠 Long-Term Memory
Can remember information for hundreds of time steps without degradation
📈 Stable Gradients
Cell state provides linear gradient path, preventing vanishing gradients
🎯 Selective Processing
Gates allow selective reading, writing, and forgetting of information
🔧 Flexible Architecture
Can be stacked, bidirectional, or used in seq2seq models
