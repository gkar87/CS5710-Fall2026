Simple Linear Regression Model

The Model

Hypothesis Function:
h(x) = θ₀ + θ₁x
Predicts output y based on input x
  • θ₀ Intercept (bias) - Where the line crosses y-axis
  • θ₁ Slope (weight) - Rate of change
  • x Feature - Input variable
  • y Target - Output we want to predict

Key Concepts

Why Linear?

We assume a linear relationship between input and output. The model learns the best straight line that fits the data.

Goal

Find optimal values of θ₀ and θ₁ that minimize the difference between predictions h(x) and actual values y.

# Python representation
def hypothesis(x, theta0, theta1):
    return theta0 + theta1 * x

# Example prediction
# If θ₀ = 50, θ₁ = 30, x = 2
# h(2) = 50 + 30*2 = 110
