Monte Carlo Methods

What are Monte Carlo Methods?

Monte Carlo (MC) methods learn from complete episodes of experience without requiring a model of the environment.
Key characteristics:

• Learn from complete episodes
• No bootstrapping (unlike TD methods)
• Model-free learning
• Based on averaging sample returns
  • 🎯 First-Visit MC
    Estimates value as average of returns following first visits to states
  • 🔄 Every-Visit MC
    Averages returns following all visits to states
  • 📊 MC Prediction
    Evaluates a given policy using sample episodes
  • ⚡ MC Control
    Finds optimal policies through exploration and exploitation

Monte Carlo Algorithm

First-Visit MC Prediction:

1. Initialize V(s) arbitrarily for all s
2. Initialize Returns(s) ← empty list
3. For each episode:
a. Generate episode: S₀, A₀, R₁, S₁, A₁, R₂, ..., Sₜ
b. G ← 0
c. For t = T-1, T-2, ..., 0:
• G ← γG + R_{t+1}
• If Sₜ not in S₀, S₁, ..., Sₜ₋₁:
- Append G to Returns(Sₜ)
- V(Sₜ) ← average(Returns(Sₜ))
Value Update:
V(s) ← V(s) + α[Gₜ - V(s)]
where Gₜ is the actual return from time t

Advantages & Characteristics

🎲
No Model Required
Learn directly from experience without environment dynamics
🎯
Unbiased Estimates
Converges to true values with sufficient samples
🔄
Episode-Based
Natural for episodic tasks with clear endings
📈
High Variance
Can have high variance but zero bias
