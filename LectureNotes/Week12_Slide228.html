PR vs ROC Curves

When to Use PR Curves
Best for imbalanced datasets where positive class is rare. More informative than ROC when you care about positive predictions.
When to Use ROC Curves
Better for balanced datasets. Good when you need to evaluate both positive and negative class performance equally.
Key Difference
PR curves focus on positive class performance, while ROC curves consider both classes. PR curves can be more pessimistic.

PR Curves for Imbalanced Datasets

🏥
Medical Diagnosis
Rare diseases (1% prevalence). PR curves show if model can handle class imbalance better than ROC.
🔍
Fraud Detection
Fraudulent transactions (0.1% of total). High precision needed to minimize false alarms.
📧
Spam Detection
Spam emails (5-15% of inbox). Balance precision vs recall to avoid blocking legitimate emails.
Key Insight:
In imbalanced datasets, a high ROC AUC can be misleading. A model with 99% accuracy might still perform poorly on the minority class!
