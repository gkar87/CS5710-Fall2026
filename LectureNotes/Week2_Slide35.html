Ridge Regression

🔵 L2 Regularization

Ridge Regression adds an L2 penalty term to linear regression, constraining the sum of squared coefficients to prevent overfitting.
J(θ) = MSE + λ∑θᵢ²
Where:
• λ (lambda) = regularization parameter
• ∑θᵢ² = sum of squared coefficients
• Higher λ → more regularization
• λ = 0 → standard linear regression

⚡ Key Advantages

🎯 Prevents Overfitting
Shrinks coefficients to reduce model complexity
🤝 Handles Multicollinearity
Distributes weights among correlated features
📐 Closed-Form Solution
θ = (XᵀX + λI)⁻¹Xᵀy
🔄 Always Invertible
λI ensures (XᵀX + λI) is invertible
