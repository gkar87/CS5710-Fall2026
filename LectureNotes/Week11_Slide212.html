Q-Learning Algorithm

Q-Learning Overview

Q-Learning is an off-policy TD control algorithm that learns the optimal action-value function Q* directly.
Key insight: Learn Q-values for the optimal policy while following any exploration policy.
Q(s,a) ← Q(s,a) + α[r + γ max Q(s',a') - Q(s,a)] a'
Off-Policy Learning:
• Behavior policy: ε-greedy for exploration
• Target policy: greedy (optimal)

Key Properties

  • 🎯 Model-Free
    No need for environment dynamics P(s'|s,a)
  • 🔄 Off-Policy
    Learns optimal policy while exploring
  • ⚡ Online
    Updates after every step
  • ✅ Convergence
    Guaranteed to converge to Q* with conditions
  • 🎲 Exploration
    ε-greedy strategy balances exploration/exploitation

Q-Learning Algorithm

Q-Learning (off-policy TD control):

# Initialize Q-values arbitrarily
Initialize Q(s,a) for all s ∈ S, a ∈ A(s), arbitrarily
Q(terminal, ·) = 0

Repeat for each episode:
  Initialize S
  Repeat for each step of episode:
    # Choose action using ε-greedy
    Choose A from S using policy derived from Q (e.g., ε-greedy)
    Take action A, observe R, S'
    # Q-Learning update (off-policy)
    Q(S,A) ← Q(S,A) + α[R + γ max Q(S',a) - Q(S,A)]
                                 a
    S ← S'
  until S is terminal
