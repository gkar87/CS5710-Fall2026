Perceptron Learning Algorithm

The Algorithm

  • Step 1: Initialize
    Initialize weights w and bias b to small random values (or zeros)
  • Step 2: For Each Sample
    For each training example (x, y), compute output ŷ = sign(w·x + b)
  • Step 3: Update Rule
    If ŷ ≠ y, update: w = w + η·y·x and b = b + η·y
  • Step 4: Repeat
    Repeat until convergence or max iterations reached
Key Properties:
  • Guaranteed to converge for linearly separable data
  • Learning rate η typically set to 1
  • Makes mistakes to learn the decision boundary

Python Implementation

import numpy as np

class Perceptron:
    def __init__(self, learning_rate=1.0, max_iter=1000):
        self.lr = learning_rate
        self.max_iter = max_iter
        self.weights = None
        self.bias = None

    def fit(self, X, y):
        n_samples, n_features = X.shape
        # Initialize weights and bias
        self.weights = np.zeros(n_features)
        self.bias = 0

        for _ in range(self.max_iter):
            errors = 0
            for xi, yi in zip(X, y):
                # Compute prediction
                y_pred = self.predict_single(xi)
                # Update if misclassified
                if yi != y_pred:
                    self.weights += self.lr * yi * xi
                    self.bias += self.lr * yi
                    errors += 1
            if errors == 0:
                break

    def predict_single(self, x):
        linear_output = np.dot(x, self.weights) + self.bias
        return 1 if linear_output >= 0 else -1
