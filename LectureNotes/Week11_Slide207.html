Bellman Equations

The Bellman equations are fundamental recursive relationships that describe the optimal value of being in a state. They form the theoretical foundation for dynamic programming in reinforcement learning.
State Value Function
Vπ(s) = E[Rt+1 + γVπ(St+1)]
Expected value of starting in state s and following policy π
Action Value Function
Qπ(s,a) = E[Rt+1 + γVπ(St+1)]
Expected value of taking action a in state s and then following policy π
Bellman Equation Intuition
🎯
Optimality Principle
Optimal solutions contain optimal sub-solutions
🔄
Recursive Structure
Value = Immediate reward + Discounted future value
⚖️
Consistency
Current value must match expected future return
