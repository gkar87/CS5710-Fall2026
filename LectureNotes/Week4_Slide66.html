Decision Trees for Regression

📈 Regression Trees Concept

Regression trees predict continuous numerical values instead of discrete classes. They split data to minimize prediction error rather than maximize information gain.
MSE = (1/n) × Σ(yi - ŷ)²

Where yi = actual value, ŷ = predicted value

🎯 How It Works

  • • Find best split that minimizes MSE
  • • Each leaf predicts the mean of its samples
  • • Continue splitting until stopping criteria
  • • Prediction = average value in leaf node

🔄 Classification vs Regression Trees

  • 📊 Splitting Criterion
    Classification: Information Gain/Gini
    Regression: MSE/MAE minimization
  • 🎯 Leaf Node Predictions
    Classification: Most common class
    Regression: Mean of target values
  • 📈 Output Type
    Classification: Discrete categories
    Regression: Continuous numbers
  • 📏 Evaluation Metrics
    Classification: Accuracy, F1-score
    Regression: MSE, MAE, R²

🔧 Regression Tree Algorithm

1
Calculate MSE
For current node, calculate Mean Squared Error of all target values
2
Find Best Split
Test all possible splits and choose one that minimizes weighted MSE of children
3
Calculate Reduction
MSE Reduction = MSE_parent - (weight_left × MSE_left + weight_right × MSE_right)
4
Split or Stop
If reduction > threshold, split. Otherwise, create leaf with mean target value
