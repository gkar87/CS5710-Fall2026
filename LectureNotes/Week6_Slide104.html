K-Means Algorithm Steps

Detailed Algorithm Steps

  1. 1
    Initialize Centroids
    Randomly select K data points as initial cluster centers
  2. 2
    Assign Points to Clusters
    Calculate distance from each point to all centroids, assign to nearest
  3. 3
    Update Centroids
    Compute new centroid as mean of all points in each cluster
  4. 4
    Check Convergence
    If centroids haven't moved (or moved less than threshold), stop
  5. 5
    Repeat
    Go back to step 2 until convergence or max iterations reached

Pseudocode Implementation

function KMeans(X, K, max_iter=100): # Step 1: Initialize centroids centroids = random_select(X, K) for iteration in 1 to max_iter: # Step 2: Assign points to clusters clusters = [] for point in X: distances = compute_distances(point, centroids) cluster_id = argmin(distances) clusters[cluster_id].append(point) # Step 3: Update centroids old_centroids = centroids.copy() for k in 1 to K: if clusters[k] is not empty: centroids[k] = mean(clusters[k]) # Step 4: Check convergence if converged(old_centroids, centroids): break return centroids, clusters

Algorithm Complexity Analysis

Time Complexity
O(n·k·i·d)
Space Complexity
O(n·d + k·d)
Iterations
Usually < 20
Convergence
Guaranteed*
*Convergence to local minimum is guaranteed, but global optimum is not
