Deep Q-Networks (DQN)

What is DQN?

Deep Q-Networks combine Q-Learning with deep neural networks to handle high-dimensional state spaces like images.
The Challenge:
• Tabular Q-learning fails with large/continuous states
• Need function approximation for real-world problems
• Neural networks can learn complex representations

The Solution:
Replace Q-table with neural network:
Q(s,a; θ) ≈ Q*(s,a)

Key Innovations

  • 🧠 Neural Network Q-Function
    Deep CNN to approximate Q-values from raw pixels
  • 📦 Experience Replay
    Store and randomly sample past experiences
  • 🎯 Target Network
    Separate network for stable TD targets
  • 🖼️ Frame Stacking
    Stack consecutive frames for motion information
  • ⚙️ Preprocessing
    Grayscale, resize, normalize for efficiency
