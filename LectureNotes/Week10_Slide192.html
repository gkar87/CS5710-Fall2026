Word Embeddings: Word2Vec & GloVe

🧠 Word2Vec

Word2Vec learns dense vector representations by predicting words from context (CBOW) or context from words (Skip-gram).
📊 Skip-gram Example
Target: "learning" Context: ["machine", "is", "fun", "and"] → Predict context words from target
🎯 CBOW Example
Context: ["machine", "is", "fun", "and"] Target: "learning" → Predict target from context words
  • 1. Initialize random word vectors
  • 2. Slide window over text corpus
  • 3. Train neural network to predict
  • 4. Extract learned word vectors

🌐 GloVe

GloVe combines global matrix factorization with local context windows to capture both global and local word statistics.
📈 Co-occurrence Matrix
Count how often words appear together: X[i,j] = frequency of word i in context of j
🔢 Objective Function
Minimize: Σ f(X[i,j]) * (w[i]ᵀw[j] + b[i] + b[j] - log(X[i,j]))²
  • 1. Build word co-occurrence matrix
  • 2. Apply weighting function f(x)
  • 3. Optimize objective function
  • 4. Combine word + context vectors

📊 Embedding Methods Comparison

🧠
Word2Vec
• Method: Neural network prediction
• Training: Local context windows
• Speed: Fast training
• Memory: Efficient
• Best for: Large corpora, analogies
🌐
GloVe
• Method: Matrix factorization
• Training: Global co-occurrence
• Speed: Medium training
• Memory: Higher usage
• Best for: Balanced global/local info
⚡
FastText
• Method: Subword information
• Training: Character n-grams
• Speed: Very fast
• Memory: Compact
• Best for: OOV words, morphology
