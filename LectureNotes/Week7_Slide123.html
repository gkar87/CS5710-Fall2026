Principal Component Analysis (PCA)

Core Concept

PCA finds the directions (principal components) along which data varies the most, then projects data onto these new axes.
Key Ideas:

• Find axes of maximum variance
• Components are orthogonal (uncorrelated)
• First PC captures most variance
• Linear transformation of data
PC₁ = arg max‖w‖=1 Var(Xw)

How It Works

  • 1
    Center the Data
    Subtract mean from each feature
  • 2
    Compute Covariance Matrix
    C = (1/n)X^T X
  • 3
    Find Eigenvectors
    Solve Cv = λv
  • 4
    Sort by Eigenvalues
    Largest eigenvalue = most variance
  • 5
    Project Data
    Y = XW (W = top k eigenvectors)
