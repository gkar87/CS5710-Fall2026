LSTM Gates: Forget, Input, Output

Three Gates Control Information Flow

LSTM uses three specialized neural networks (gates) to control what information flows through the cell state. Each gate outputs values between 0 and 1, acting as learned switches.

ft
Forget Gate
Decides what information to discard from the cell state
ft = σ(Wf · [ht-1, xt] + bf)
Function: "What old information is no longer relevant?"
Output: 0 = completely forget, 1 = completely keep
Applied to: Previous cell state Ct-1
it
Input Gate
Determines what new information to store in the cell state
it = σ(Wi · [ht-1, xt] + bi)
C̃t = tanh(WC · [ht-1, xt] + bC)
Function: "What new information should we store?"
Output: 0 = ignore new info, 1 = store all new info
Applied to: Candidate values C̃t
ot
Output Gate
Controls what parts of cell state to output as hidden state
ot = σ(Wo · [ht-1, xt] + bo)
ht = ot ⊙ tanh(Ct)
Function: "What stored information is relevant now?"
Output: 0 = hide from output, 1 = include in output
Applied to: Current cell state Ct

Step 4: Output
