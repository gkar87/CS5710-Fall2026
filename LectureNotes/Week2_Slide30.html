Feature Scaling and Normalization

Why Scale Features?

Feature Scaling ensures all features contribute equally to the learning algorithm by transforming them to similar scales, preventing features with larger values from dominating the model.
Problem: Features with different scales cause issues:

• House prices: $100,000 - $500,000
• Square footage: 800 - 3,000
• Number of bedrooms: 1 - 5

Without scaling, price dominates because its values are much larger!

Scaling Methods

Min-Max Scaling
Scales features to [0,1] range
X' = (X - X_min) / (X_max - X_min)
Z-Score (StandardScaler)
Centers around 0 with unit variance
X' = (X - μ) / σ
Robust Scaling
Uses median and IQR, handles outliers
X' = (X - median) / IQR
