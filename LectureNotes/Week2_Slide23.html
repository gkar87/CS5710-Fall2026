Cost Function (MSE)

Mean Squared Error

Cost Function J(θ₀, θ₁):
J(θ₀, θ₁) = 1⁄2m Σ(h(x⁽ⁱ⁾) - y⁽ⁱ⁾)²
Measures average squared difference between predictions and actual values
Where:
• m = number of training examples
• h(x⁽ⁱ⁾) = predicted value for example i
• y⁽ⁱ⁾ = actual value for example i
• 1⁄2 factor makes math cleaner later

Why Square the Error?

  • 🎯 Penalizes Large Errors
    Squaring amplifies larger errors more than small ones
  • ➕ Always Positive
    Prevents positive and negative errors from canceling out
  • 📈 Differentiable
    Smooth function makes optimization easier
  • 🔧 Convex Function
    Has a single global minimum (no local minima)
Goal: Find θ₀ and θ₁ that minimize J(θ₀, θ₁)

min J(θ₀, θ₁)

Understanding MSE

Step 1: Calculate error for each point
error⁽ⁱ⁾ = h(x⁽ⁱ⁾) - y⁽ⁱ⁾
Step 2: Square each error
(error⁽ⁱ⁾)²
Step 3: Average all squared errors
1⁄m Σ(error⁽ⁱ⁾)²
Example with 5 points:
Errors: [2, -3, 1, -2, 4]
Squared: [4, 9, 1, 4, 16]
MSE = (4+9+1+4+16)/5 = 6.8
