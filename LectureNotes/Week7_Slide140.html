🔍 Dimensionality Reduction Methods Comparison

Method Type Preserves Complexity Best For
PCA Linear Variance O(n³) Fast, interpretable reduction
t-SNE Non-linear Local structure O(n²) Visualization (2D/3D)
UMAP Non-linear Global + Local O(n¹·⁴) Fast visualization, clustering
Autoencoders Non-linear Reconstruction O(n·epochs) Complex patterns, generation
LDA Linear Class separation O(n³) Supervised reduction
