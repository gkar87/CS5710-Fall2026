k-NN for Regression

From Classification to Regression

k-NN Regression adapts the nearest neighbor concept to predict continuous values instead of discrete classes. Instead of voting, we use mathematical functions to combine neighbor values.
Key Differences:

• Output: Continuous numerical value, not class label
• Combination: Mathematical functions (mean, weighted average)
• Evaluation: MSE, MAE, R² instead of accuracy
• Smoothness: Creates smooth prediction surfaces

Aggregation Methods

📊 Simple Average
ŷ = (1/k) × Σyᵢ
Equal weight to all k neighbors. Simple but ignores distance information.
⚖️ Weighted Average
ŷ = Σ(wᵢ × yᵢ) / Σwᵢ
Closer neighbors get higher weights. More accurate but computationally intensive.
🎯 Distance Weighting
wᵢ = 1/dᵢ² or wᵢ = e^(-dᵢ)
Common weight functions based on inverse distance or Gaussian kernel.
