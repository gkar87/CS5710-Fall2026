Sigmoid Function and Decision Boundary

The Sigmoid Function

The sigmoid function (also called logistic function) maps any real-valued number to the range (0, 1). It's the key component that makes logistic regression a classifier.
σ(z) = 1 / (1 + e^(-z))

Decision Boundary

The decision boundary is the surface that separates different classes. For logistic regression with threshold 0.5, it's where σ(z) = 0.5, i.e., z = 0.
θ^T x = 0
Decision boundary equation
Prediction Rule:
• If θ^T x ≥ 0: predict y = 1
• If θ^T x < 0: predict y = 0
This creates a linear boundary!

Understanding the Boundary

  • 1
    The probability threshold
    σ(z) = 0.5 when z = 0
  • 2
    Linear decision boundary
    θ_0 + θ_1 x_1 + θ_2 x_2 = 0
  • 3
    Classification regions
    θ^T x > 0 → Class 1
    θ^T x < 0 → Class 0
  • 4
    Nonlinear boundaries with polynomial features
    θ_0 + θ_1 x + θ_2 x² + ... = 0
