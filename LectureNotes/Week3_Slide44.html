Sigmoid Function & Decision Boundary

🎯 Sigmoid Function

The sigmoid function is the heart of logistic regression. It maps any real number to a value between 0 and 1, making it perfect for probability estimation.
σ(z) = 1 / (1 + e^(-z))

Where z = θ₀ + θ₁x₁ + θ₂x₂ + ... + θₙxₙ
  • 📈 S-Shaped Curve
    Smooth transition from 0 to 1
  • ⚖️ Threshold at 0.5
    When z = 0, σ(z) = 0.5
  • 📊 Output Range [0,1]
    Perfect for probability interpretation
  • 🔄 Differentiable
    Enables gradient descent optimization

📏 Decision Boundary

The decision boundary is where the model makes classification decisions. It's the line (or surface) where the probability equals 0.5.
Classification Rules:

• If σ(z) ≥ 0.5 → Predict Class 1
• If σ(z) < 0.5 → Predict Class 0

Decision Boundary: z = 0
θ₀ + θ₁x₁ + θ₂x₂ = 0
Key Insight: The decision boundary is linear in the feature space, but the sigmoid function creates a smooth probabilistic transition across this boundary.
