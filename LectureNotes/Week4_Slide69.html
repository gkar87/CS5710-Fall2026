Feature Importance in Random Forests

🎯 What is Feature Importance?

Feature importance measures how much each feature contributes to decreasing node impurity across all trees in the forest. Higher importance indicates greater predictive power.
Key Benefits:

• Feature Selection - Remove irrelevant features
• Model Interpretation - Understand what drives predictions
• Domain Insights - Discover important patterns in data
• Debugging - Identify unexpected feature influences

📊 Calculation Methods

🌳 Mean Decrease Impurity (MDI)
Default sklearn method. Averages impurity decrease across all trees where the feature is used for splitting.
📈 Mean Decrease Accuracy (MDA)
Measures accuracy drop when feature values are randomly permuted. More robust but computationally expensive.
🎲 SHAP Values
Game theory approach providing consistent, additive feature attributions for individual predictions.
