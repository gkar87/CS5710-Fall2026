LIME Explanations

LIME Algorithm Steps

1
Select Instance to Explain
Choose the specific prediction you want to understand and explain to stakeholders.
2
Generate Perturbations
Create many variations of the instance by randomly modifying features while staying realistic.
3
Get Model Predictions
Run the black-box model on all perturbed samples to see how predictions change.
4
Weight by Similarity
Assign higher weights to perturbations that are more similar to the original instance.
5
Fit Local Linear Model
Train a simple, interpretable model (like linear regression) on the weighted perturbed data.
6
Extract Explanation
Use the linear model's coefficients as feature importance scores for the local explanation.
