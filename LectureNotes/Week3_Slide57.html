Gaussian Naive Bayes

Handling Continuous Features

Gaussian Naive Bayes extends Naive Bayes to handle continuous (numerical) features by assuming they follow a normal (Gaussian) distribution within each class.
📈 Continuous Features
Height, weight, temperature, price - assumed to follow Gaussian distribution
🔢 Discrete Features
Still handled categorically (e.g., color: red/blue/green)
🔄 Mixed Features
Can handle both continuous and discrete features in the same model
Key Advantage: No need to discretize continuous features - maintains all information and natural relationships.

Gaussian Probability Density

P(x|class) = (1/√(2πσ²)) × e^(-(x-μ)²/(2σ²))

Parameter Estimation

μ (Mean)
μ = Σx / n
σ² (Variance)
σ² = Σ(x - μ)² / n
σ (Standard Deviation)
σ = √(σ²)
These parameters are calculated separately for each feature within each class

Distribution Parameters
