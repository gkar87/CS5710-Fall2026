Voting Classifiers

🗳️
Hard Voting
ŷ = mode{h₁(x), h₂(x), ..., hₘ(x)}
Majority Rule: Each classifier votes for a class, and the class with the most votes wins. Simple democratic approach.

Example: If 3 classifiers predict [Cat, Dog, Cat], the final prediction is Cat (2 votes vs 1).
📊
Soft Voting
ŷ = argmax Σᵢ pᵢ(class)
Probability Average: Averages class probabilities from all classifiers, then picks the class with highest average probability.

Example: Average probabilities [Cat: 0.7, Dog: 0.3] across all models, then predict the class with max average.

Implementation Example

# Voting Classifier Implementation from sklearn.ensemble import VotingClassifier from sklearn.svm import SVC from sklearn.ensemble import RandomForestClassifier from sklearn.linear_model import LogisticRegression from sklearn.naive_bayes import GaussianNB # Define base classifiers classifiers = [ ('svm', SVC(probability=True)), # Need probability=True for soft voting ('rf', RandomForestClassifier()), ('lr', LogisticRegression()), ('nb', GaussianNB()) ] # Hard Voting Classifier hard_voting_clf = VotingClassifier( estimators=classifiers, voting='hard' # Majority vote ) # Soft Voting Classifier (usually performs better) soft_voting_clf = VotingClassifier( estimators=classifiers, voting='soft' # Average probabilities ) # Train the ensemble hard_voting_clf.fit(X_train, y_train) soft_voting_clf.fit(X_train, y_train) # Make predictions hard_predictions = hard_voting_clf.predict(X_test) soft_predictions = soft_voting_clf.predict(X_test) # Compare performance print(f"Hard Voting Accuracy: {hard_voting_clf.score(X_test, y_test):.3f}") print(f"Soft Voting Accuracy: {soft_voting_clf.score(X_test, y_test):.3f}")
