Dendrograms

🌳 What is a Dendrogram?

A dendrogram is a tree-like diagram that shows the arrangement of clusters produced by hierarchical clustering. It visualizes the step-by-step merging process and the distances at which clusters combine.
  • 📏 Shows merge distances/heights
  • ✂️ Can be cut at any level for clustering
  • 🔍 Reveals hierarchical structure
  • 📊 Helps determine optimal clusters

🎯 Reading Dendrograms

Height Interpretation
The y-axis shows the distance or dissimilarity at which clusters merge. Higher values indicate greater dissimilarity between merged clusters.
Cutting the Tree
Draw a horizontal line across the dendrogram. The number of vertical lines it intersects determines the number of clusters.
Cluster Quality
Large jumps in height suggest natural cluster boundaries. Small incremental increases indicate gradual merging.
Optimal Clusters
Look for the "elbow" - where the height increases dramatically. This suggests the optimal number of clusters.
