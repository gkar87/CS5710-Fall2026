🌀 Kernel PCA

🎯 The Kernel Trick

Key Idea: Project data into higher-dimensional space where it becomes linearly separable, without explicitly computing the transformation.

Why Kernel PCA?

  • • Handle non-linear relationships
  • • Capture complex patterns
  • • No explicit feature mapping needed
  • • Works in infinite-dimensional spaces
Mathematical Foundation:
Instead of φ(x)ᵀφ(y), compute k(x,y) directly
where k is the kernel function

📐 Common Kernels

Polynomial Kernel
k(x,y) = (xᵀy + c)ᵈ
Captures polynomial features
RBF (Gaussian) Kernel
k(x,y) = exp(-γ||x-y||²)
Most popular, infinite dimensions
Sigmoid Kernel
k(x,y) = tanh(αxᵀy + c)
Similar to neural networks
Laplacian Kernel
k(x,y) = exp(-α||x-y||)
Robust to outliers

💡 Advantages & Considerations

✅
Advantages
  • • Handles non-linear patterns
  • • No need to specify features
  • • Works with any data type
  • • Powerful for complex datasets
⚠️
Considerations
  • • Computationally expensive O(n³)
  • • Kernel selection is crucial
  • • Parameter tuning required
  • • Difficult to interpret components
