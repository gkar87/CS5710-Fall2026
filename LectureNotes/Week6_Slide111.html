DBSCAN Algorithm

🎯 What is DBSCAN?

Density-Based Spatial Clustering of Applications with Noise (DBSCAN) is a density-based clustering algorithm that groups together points in high-density areas and marks points in low-density areas as outliers.
  • 🔢 No need to specify number of clusters
  • 🌟 Finds clusters of arbitrary shape
  • 🚫 Robust to outliers and noise
  • 🎯 Based on density reachability

⚙️ Algorithm Steps

  1. Initialize: Mark all points as unvisited
  2. For each point P: If P is unvisited, mark as visited
  3. Find neighbors: Get all points within ε distance of P
  4. Check density: If neighbors < minPts, mark P as noise
  5. Create cluster: If neighbors ≥ minPts, start new cluster
  6. Expand cluster: Add all density-reachable points

⚖️ Advantages vs Disadvantages

✅ Advantages
  • No need to specify number of clusters beforehand
  • Can find clusters of arbitrary shapes and sizes
  • Robust to outliers - identifies noise points
  • Deterministic results (unlike K-means)
  • Works well with varying cluster densities
  • Can discover clusters with non-convex shapes
❌ Disadvantages
  • Sensitive to hyperparameters (ε and minPts)
  • Struggles with varying densities
  • Can be computationally expensive O(n²)
  • Difficult to use with high-dimensional data
  • Memory usage can be high for large datasets
  • Border points may be assigned to different clusters
