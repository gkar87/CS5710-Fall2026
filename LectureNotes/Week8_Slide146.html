Forward Propagation

Neural Network Flow

Click to see the signal flow through the network

Forward Pass Steps

  • 1
    Input Layer
    Receive input features x₁, x₂, ..., xₙ
  • 2
    Weighted Sum
    Calculate z = Σ(wᵢxᵢ) + b for each neuron
  • 3
    Activation
    Apply activation function a = f(z)
  • 4
    Propagate
    Pass output to next layer as input
  • 5
    Output
    Generate final prediction ŷ

Mathematical Formulation

Layer l computation:
z[l] = W[l] · a[l-1] + b[l]
a[l] = g[l](z[l])

Where:
W[l] = weight matrix for layer l
b[l] = bias vector for layer l
g[l] = activation function for layer l
a[0] = x (input features)
