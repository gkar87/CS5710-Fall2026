Backpropagation Through Time (BPTT)

What is BPTT?

Backpropagation Through Time is the gradient computation method for training Recurrent Neural Networks. It extends standard backpropagation by unrolling the RNN through time steps.

Key Insight: RNNs share parameters across all time steps, so we need to accumulate gradients from all time steps to update the weights.

Gradient Accumulation:
∂L/∂W = Σt=1T ∂Lt/∂W

Where:
• L = Total loss over sequence
• Lt = Loss at time step t
• W = Shared weight parameters
• T = Total number of time steps

BPTT Algorithm Steps

  • 1
    Forward Pass
    Process input sequence through all time steps, storing all hidden states and outputs
  • 2
    Compute Loss
    Calculate loss at each time step and sum for total sequence loss
  • 3
    Backward Pass
    Propagate gradients backward through time, from T to 1
  • 4
    Accumulate Gradients
    Sum gradients across all time steps for shared parameters
  • 5
    Update Weights
    Apply accumulated gradients to update network parameters
