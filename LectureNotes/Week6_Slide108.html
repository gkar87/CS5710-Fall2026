Agglomerative vs Divisive Clustering

⬆️ Agglomerative (Bottom-Up)

Starts with individual points and progressively merges them into larger clusters until reaching one root cluster.

⬇️ Divisive (Top-Down)

Starts with all points in one cluster and recursively splits them into smaller clusters until each point is isolated.

Direct Comparison

Aspect Agglomerative Divisive
Starting Point N clusters (individual points) 1 cluster (all points)
Process Merge closest clusters Split clusters optimally
Time Complexity O(n² log n) O(2ⁿ) worst case
Best For Small, detailed clusters Large, well-separated clusters
Implementation Simpler, widely available Complex, less common
