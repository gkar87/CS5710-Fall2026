Linear Discriminant Analysis (LDA)

What is LDA?

🎯 Supervised Dimensionality Reduction
LDA finds linear combinations of features that best separate classes
📊 Gaussian Assumptions
Assumes normally distributed data with equal covariance matrices
J(w) = (w^T S_B w) / (w^T S_W w)
🔍 Maximize Class Separation
Maximizes between-class variance while minimizing within-class variance

LDA vs PCA

Aspect LDA PCA
Supervision Supervised Unsupervised
Goal Maximize class separation Maximize variance
Components At most C-1 (C = classes) Min(n_samples, n_features)
Use Case Classification preprocessing General dimensionality reduction

LDA Algorithm Steps

1
Compute Class Means
Calculate mean vectors for each class in the dataset
2
Calculate Scatter Matrices
Compute within-class (S_W) and between-class (S_B) scatter matrices
3
Solve Eigenvalue Problem
Find eigenvectors of S_W^(-1)S_B
4
Transform Data
Project data onto selected discriminant vectors
