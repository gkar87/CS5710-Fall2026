Gaussian Mixture Models (GMM)

🎯 Core Concept

Gaussian Mixture Model (GMM) is a probabilistic clustering algorithm that models data as a mixture of multiple Gaussian distributions. Each cluster is represented by a Gaussian distribution with its own parameters (mean, covariance, mixing coefficient).
  • 📊 Probabilistic clustering approach
  • 🎲 Soft assignment of points to clusters
  • 📈 Models elliptical cluster shapes
  • 🎯 Estimates cluster parameters
  • 🔧 Uses Expectation-Maximization (EM)

⚖️ GMM vs Other Clustering Methods

Aspect GMM K-Means DBSCAN
Assignment Soft (probabilistic) Hard (binary) Hard (density-based)
Cluster Shape Elliptical (flexible) Spherical (circular) Arbitrary shape
Parameter Selection Number of components (K) Number of clusters (k) ε (epsilon), minPts
Probabilistic Output Yes (cluster probabilities) No (only assignments) No (binary: cluster or noise)
Handling Outliers Moderate (soft assignment) Poor (outliers affect centroids) Excellent (explicit noise detection)
Computational Complexity O(ndk) per iteration O(ndk) per iteration O(n²) worst case
