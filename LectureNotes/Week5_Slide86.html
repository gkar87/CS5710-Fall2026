Gradient Boosting Algorithm

Core Concept

Gradient Boosting builds models sequentially, where each new model corrects the errors of the previous ensemble by fitting to the residuals.
Key principles:

• Sequential Learning: Models added one at a time
• Error Correction: Each model focuses on previous mistakes
• Gradient Descent: Optimizes loss function in function space
• Weak Learners: Typically shallow decision trees
• Additive Model: Final prediction is sum of all models

Algorithm Steps

  • Initialize with constant prediction
    F₀(x) = argmin_γ Σ L(yᵢ, γ)
  • Compute pseudo-residuals
    rᵢₘ = -[∂L(yᵢ, F(xᵢ))/∂F(xᵢ)]
  • Fit weak learner to residuals
    Train hₘ(x) on {(xᵢ, rᵢₘ)}
  • Find optimal step size
    γₘ = argmin_γ Σ L(yᵢ, Fₘ₋₁(xᵢ) + γhₘ(xᵢ))
  • Update model
    Fₘ(x) = Fₘ₋₁(x) + γₘhₘ(x)

Mathematical Foundation

F(x) = F₀(x) + Σ(m=1 to M) γₘ·hₘ(x)
