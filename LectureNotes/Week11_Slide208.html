Dynamic Programming in RL

Dynamic Programming (DP) methods solve reinforcement learning problems by iteratively applying the Bellman equations when we have perfect knowledge of the environment dynamics.
📊 Value Iteration
  1. 1 Initialize V(s) for all states
  2. 2 For each state, update V(s) using Bellman optimality equation
  3. 3 Repeat until convergence
  4. 4 Extract optimal policy from V*
🎯 Policy Iteration
  1. 1 Initialize policy π randomly
  2. 2 Evaluate policy: compute V^π
  3. 3 Improve policy: π' = greedy(V^π)
  4. 4 Repeat until policy stable
Key Properties of Dynamic Programming
✅
Guaranteed Convergence
Always finds optimal solution
🔄
Iterative Improvement
Each iteration improves the solution
📋
Model Required
Needs full environment knowledge
