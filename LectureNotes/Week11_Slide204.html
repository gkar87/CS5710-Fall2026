Value Functions and Bellman Equations

Value Functions

🎯 State Value Function V^π(s)
V^π(s) = E_π[G_t | S_t = s]
Expected return starting from state s following policy π
⚡ Action Value Function Q^π(s,a)
Q^π(s,a) = E_π[G_t | S_t = s, A_t = a]
Expected return starting from state s, taking action a, then following policy π
⭐ Optimal Value Functions
V*(s) = max_π V^π(s)
Q*(s,a) = max_π Q^π(s,a)
Maximum possible value achievable from state s (or state-action pair)

Key Relationships

Value Function Relationships:

V^π(s) = Σ_a π(a|s) Q^π(s,a)

Q^π(s,a) = Σ_{s'} P(s'|s,a)[R(s,a,s') + γV^π(s')]

Optimal Relationships:

V*(s) = max_a Q*(s,a)

Q*(s,a) = Σ_{s'} P(s'|s,a)[R(s,a,s') + γV*(s')]
Bellman Equations
Bellman Expectation Equation
V^π(s) = Σ_a π(a|s) Σ_{s'} P(s'|s,a)[R(s,a,s') + γV^π(s')]
Expected value under policy π
Bellman Optimality Equation
V*(s) = max_a Σ_{s'} P(s'|s,a)[R(s,a,s') + γV*(s')]
Optimal value from state s
Q-Function Bellman Equation
Q^π(s,a) = Σ_{s'} P(s'|s,a)[R(s,a,s') + γΣ_{a'} π(a'|s')Q^π(s',a')]
Action-value expectation
Optimal Q-Function
Q*(s,a) = Σ_{s'} P(s'|s,a)[R(s,a,s') + γ max_{a'} Q*(s',a')]
Optimal action-value
