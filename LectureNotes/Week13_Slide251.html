🔍 Anomaly Detection Techniques

🛠️ Core Techniques

📊 Statistical Methods
Z-score, IQR, Grubbs' test for identifying outliers based on statistical distributions
🌳 Isolation Forest
Isolates anomalies by randomly selecting features and split values
🎯 One-Class SVM
Learns the boundary of normal data to identify outliers
📍 Local Outlier Factor
Measures local density deviation of data points
🧠 Autoencoders
Neural networks that detect anomalies through reconstruction error
🔗 DBSCAN
Density-based clustering that identifies noise points as anomalies
