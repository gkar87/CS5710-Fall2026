Temporal Difference Learning

What is TD Learning?

Temporal Difference (TD) learning combines ideas from Monte Carlo and Dynamic Programming to learn directly from experience without a model.
  • 🔄 Bootstrapping
    Updates estimates based on other estimates
  • ⚡ Online Learning
    Can learn before episode ends
  • 📊 Model-Free
    No need for environment dynamics
  • 🎯 TD(0)
    One-step lookahead version
V(Sₜ) ← V(Sₜ) + α[Rₜ₊₁ + γV(Sₜ₊₁) - V(Sₜ)]

TD vs MC vs DP

Aspect TD Monte Carlo Dynamic Prog

TD(0) Algorithm

TD(0) Prediction:

Initialize V(s) arbitrarily
Repeat for each episode:
  Initialize S
  Repeat for each step:
    A ← action from π for S
    Take action A, observe R, S'
    V(S) ← V(S) + α[R + γV(S') - V(S)]
    S ← S'
  until S is terminal
