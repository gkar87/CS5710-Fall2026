Logistic Regression Cost Function

💰 The Cost Function

The logistic regression cost function is derived from maximum likelihood estimation. It measures how well our model's predictions match the actual labels.
J(θ) = -1/m × Σᵢ₌₁ᵐ [yᵢlog(hθ(xᵢ)) + (1-yᵢ)log(1-hθ(xᵢ))]

where hθ(x) = σ(θᵀx) = 1/(1+e^(-θᵀx))
When y = 1:
Cost = -log(hθ(x))
High cost if hθ(x) ≈ 0
Low cost if hθ(x) ≈ 1
When y = 0:
Cost = -log(1-hθ(x))
High cost if hθ(x) ≈ 1
Low cost if hθ(x) ≈ 0

🧠 Intuition

Why This Cost Function?

• Convex: Single global minimum
• Differentiable: Enables gradient descent
• Penalizes wrong predictions: Cost → ∞ as prediction → wrong
• Probabilistic: Based on likelihood principle
Key Insight: The cost function severely penalizes confident wrong predictions while rewarding confident correct predictions.
Example:
• If y=1 and hθ(x)=0.9 → Cost ≈ 0.1
• If y=1 and hθ(x)=0.1 → Cost ≈ 2.3
• If y=1 and hθ(x)=0.01 → Cost ≈ 4.6
Cost Function Comparison
Linear Regression MSE
J(θ) = 1/2m × Σ(hθ(xᵢ) - yᵢ)²
❌ Problems for Classification:
• Non-convex when used with sigmoid
• Multiple local minima
• Poor gradient descent convergence
• Not probabilistically motivated
Logistic Regression
J(θ) = -1/m × Σ[y log(hθ(x)) + (1-y)log(1-hθ(x))]
✅ Advantages:
• Convex function (single minimum)
• Probabilistically motivated (MLE)
• Smooth gradients
• Guaranteed global optimum
