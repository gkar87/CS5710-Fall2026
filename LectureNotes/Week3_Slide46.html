Logistic Regression Cost Function

The Cost Function

The log loss (cross-entropy) cost measures how well the model predicts. It's based on the negative log-likelihood from MLE, transformed into a minimization problem.
J(θ) = -1/m Σ[yᵢ log(h_θ(xᵢ)) + (1-yᵢ)log(1-h_θ(xᵢ))]
Cross-entropy / Log loss
Note: MSE doesn't work well for classification because it produces a non-convex optimization surface for logistic regression!

Why Not MSE?

✅
Cross-Entropy
  • 🔔 Convex for logistic
  • 📈 Good gradients
  • 🎯 Probabilistic basis
  • 📉 Logistic-friendly
⚠️
MSE Issues
  • 🚫 Non-convex
  • ⏸️ Poor gradients
  • 📉 Sigmoid saturation
  • 🔄 Local minima

Understanding Log Loss

  • 1
    Error when prediction is wrong
    y=1, h_θ→0: cost → -log(0) → ∞
  • 2
    Error when prediction is correct
    y=1, h_θ→1: cost → -log(1) → 0
  • 3
    Asymmetric penalization
    Wrong high-confidence: very high cost
    Correct high-confidence: very low cost
  • 4
    Convex optimization
    J(θ) is convex → global minimum exists
