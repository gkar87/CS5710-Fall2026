Normal Equation

Analytical Solution

The Normal Equation provides a direct, analytical solution to linear regression without iteration. It finds the optimal parameters in one step by solving the closed-form equation.
θ = (XTX)-1XTy
The closed-form solution for optimal linear regression parameters
Where:
• θ = optimal parameters (vector)
• X = design matrix (features)
• XT = transpose of X
• (XTX)-1 = inverse matrix
• y = target values (vector)

vs Gradient Descent

🏃‍♂️
Gradient Descent
  • 🔄 Iterative process
  • ⏱️ Requires many steps
  • 🎛️ Needs learning rate
  • 📈 Works with large datasets
  • 🔧 Hyperparameter tuning
⚡
Normal Equation
  • 📐 Direct calculation
  • 🎯 One-step solution
  • 🚫 No hyperparameters
  • ⚠️ Expensive for large data
  • ✨ Exact answer

Mathematical Derivation

How do we derive the Normal Equation? Let's minimize the cost function analytically:
  • 1
    Start with the cost function (Mean Squared Error)
    J(θ) = ½(Xθ - y)T(Xθ - y)
  • 2
    Expand the matrix multiplication
    J(θ) = ½(θTXTXθ - 2θTXTy + yTy)
  • 3
    Take the derivative with respect to θ
    ∇J(θ) = XTXθ - XTy
  • 4
    Set the derivative equal to zero for minimum
    XTXθ - XTy = 0
  • 5
    Solve for θ (assuming XTX is invertible)
    θ = (XTX)-1XTy

✅ Advantages

  • ⚡
    No need to choose learning rate
  • 🎯
    Gives exact analytical solution
  • 🚫
    No iterations required
  • 📊
    Works well for small to medium datasets
  • 🔧
    No hyperparameter tuning

❌ Disadvantages

  • 💻
    O(n³) computational complexity
  • 📈
    Doesn't scale well with large datasets
  • ⚠️
    Requires XTX to be invertible
  • 💾
    Memory intensive for large feature sets
  • 🚫
    Only works for linear regression
