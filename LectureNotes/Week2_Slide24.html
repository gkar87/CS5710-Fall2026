Gradient Descent Algorithm

🎯 The Goal

Gradient Descent is an iterative optimization algorithm used to find the minimum of a function by moving in the direction of steepest descent.
θj := θj - α ∂/∂θj J(θ)
Think of it like rolling a ball down a hill:

• 🏔️ The hill is our cost function
• ⚽ The ball is our current parameter values
• ⬇️ The ball rolls toward the lowest point
• 🎯 We want to reach the global minimum

📋 Algorithm Steps

1. Initialize parameters θ randomly 2. Repeat until convergence: a. Compute cost J(θ) b. Compute gradients ∇J(θ) c. Update: θ := θ - α∇J(θ) d. Check stopping criteria 3. Return optimized θ
Key Components:

α (alpha): Learning rate - controls step size
∇J(θ): Gradient - direction of steepest ascent
θ: Parameters we're optimizing
