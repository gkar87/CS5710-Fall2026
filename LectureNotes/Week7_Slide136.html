Filter Methods

What are Filter Methods?

Filter Methods evaluate features independently using statistical measures, without considering any specific machine learning algorithm. They act as a "filter" to remove irrelevant features before training.
🔍 Key Characteristics
⚡ Fast: No model training needed
📊 Statistical: Uses mathematical measures
🔄 Reusable: Algorithm-independent
📈 Scalable: Works with large datasets

Filter Process

1
Calculate Scores
Compute statistical measure for each feature
2
Rank Features
Sort features by their scores
3
Apply Threshold
Select top-k or above-threshold features
4
Use in Model
Train any ML algorithm with selected features

Common Filter Methods

📊
Correlation
Measures linear relationship between feature and target variable
r = cov(X,y) / (σ_X × σ_y)
χ²
Chi-Square
Tests independence between categorical features and target
χ² = Σ((O-E)² / E)
ℹ️
Mutual Information
Measures how much information one variable provides about another
I(X;Y) = H(X) - H(X|Y)
📈
F-Score
ANOVA F-test for measuring linear dependency
F = MSB / MSW
📉
Variance Threshold
Removes features with low variance (near-constant values)
Var(X) = E[(X-μ)²]
🎯
Relief
Estimates feature quality based on how well features distinguish between instances
W[A] += diff(A,R,H) - diff(A,R,M)
