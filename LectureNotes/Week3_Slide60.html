Key Takeaways

Classification predicts discrete labels. Different methods suit different problems. Consider class balance and evaluation metrics!
  • ✓
    Logistic Regression: Probabilistic, interpretable
  • ✓
    k-NN: Simple, no training, slow inference
  • ✓
    Naive Bayes: Fast, works well for text
  • ✓
    F1 Score: Better than accuracy for imbalance
