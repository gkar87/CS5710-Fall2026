⚡ The Perceptron Model

Perceptron Architecture

  • 📥 Inputs (x₁, x₂, ..., xₙ)
    Feature values from the data
  • ⚖️ Weights (w₁, w₂, ..., wₙ)
    Importance of each input feature
  • ➕ Bias (b)
    Threshold adjustment term
  • 🎯 Activation Function
    Step function for binary output

Mathematical Formulation

// Linear Combination
z = Σ(wᵢ × xᵢ) + b

// Activation (Step Function)
y = {
  1, if z ≥ 0
  0, if z < 0
}

// Compact Form
y = sign(w·x + b)

Key Properties

  • ✓ Linear decision boundary
  • ✓ Binary classification only
  • ✓ Single layer architecture
  • ✓ Can learn linearly separable patterns
  • ✗ Cannot solve XOR problem
