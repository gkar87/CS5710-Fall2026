Fairness Metrics

Key Fairness Metrics

📊 Demographic Parity
Equal positive prediction rates across groups
⚖️ Equalized Odds
Equal true positive & false positive rates
🎯 Equal Opportunity
Equal true positive rates for qualified individuals
📏 Calibration
Equal prediction accuracy across groups
👤 Individual Fairness
Similar predictions for similar individuals

Mathematical Formulations

# Demographic Parity P(Ŷ=1|A=0) = P(Ŷ=1|A=1) # Equalized Odds P(Ŷ=1|Y=y,A=0) = P(Ŷ=1|Y=y,A=1) ∀y∈{0,1} # Equal Opportunity P(Ŷ=1|Y=1,A=0) = P(Ŷ=1|Y=1,A=1) # Calibration P(Y=1|Ŷ=p,A=0) = P(Y=1|Ŷ=p,A=1) = p Where: - Ŷ: Model prediction - Y: True label - A: Protected attribute - p: Predicted probability

Trade-offs

⚠️ Impossibility Theorem:

It's mathematically impossible to satisfy all fairness metrics simultaneously unless the base rates are equal across groups.
