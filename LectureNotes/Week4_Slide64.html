Building Decision Trees (ID3, CART)

🌳 Tree Building Algorithms

📊 ID3 (Iterative Dichotomiser 3)
Uses Information Gain to select the best attribute for splitting at each node.
  • Uses entropy to measure node impurity
  • Only handles categorical attributes
  • Prone to overfitting
  • Cannot handle missing values
🎯 CART (Classification & Regression Trees)
Uses Gini Impurity for classification and MSE for regression.
  • Handles both categorical and numerical data
  • Creates binary splits only
  • Built-in pruning mechanisms
  • Can handle missing values

📊 Algorithm Comparison

Aspect ID3 CART
Splitting Criteria Information Gain Gini Impurity / MSE
Data Types Categorical only Both categorical & numerical
Split Type Multi-way splits Binary splits only
Missing Values Cannot handle Surrogate splits
Pruning No built-in pruning Cost-complexity pruning
Use Case Simple categorical problems General-purpose trees

🔧 Decision Tree Building Process

1
Select Root
Choose the attribute that best separates the data using information gain or Gini impurity
2
Create Branches
Create branches for each possible value of the selected attribute
3
Split Data
Partition training examples into subsets based on attribute values
4
Recurse
Recursively apply the process to each subset until stopping criteria are met
