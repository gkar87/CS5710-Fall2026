📚 Historical Significance

LeNet-5 was developed by Yann LeCun and his team at Bell Labs in 1998. It was primarily designed for handwritten digit recognition and achieved remarkable success on the MNIST dataset.

This architecture laid the foundation for modern CNNs and demonstrated the power of:

  • Convolutional layers for local feature extraction
  • Subsampling (pooling) for translation invariance
  • Gradient-based learning for automatic feature discovery
  • End-to-end trainable systems

🎯 C1 - First Convolutional Layer

C1 is the first convolutional layer in LeNet-5:

  • Input: 32×32×1 grayscale image
  • Filters: 6 filters of size 5×5
  • Output: 28×28×6 feature maps
  • Parameters: (5×5+1)×6 = 156 parameters
  • Activation: Tanh function

🎬 C1 Layer Animation

🔍 Key Features:

  • Edge Detection: Learns basic edge and corner detectors
  • Local Connectivity: Each neuron sees only 5×5 region
  • Parameter Sharing: Same filter weights across entire image
  • Translation Equivariance: Features detected regardless of position
×

🏊 S2 - First Subsampling Layer

S2 performs subsampling (early pooling):

  • Input: 28×28×6 from C1
  • Operation: 2×2 average pooling with stride 2
  • Output: 14×14×6 feature maps
  • Parameters: 2×6 = 12 (weight + bias per map)
  • Function: σ(w × average + b)

💡 Original Implementation

Unlike modern max pooling, LeNet-5 used trainable subsampling:

  • Each 2×2 region was averaged
  • Result multiplied by learnable weight
  • Bias added and sigmoid applied
  • This allowed the network to learn optimal downsampling

🎯 Benefits:

  • Dimension Reduction: Reduces spatial size by 4x
  • Translation Invariance: Small shifts become irrelevant
  • Computational Efficiency: Fewer parameters in next layer
  • Overfitting Prevention: Acts as regularization
×

🚀 LeNet-5 Innovations

LeNet-5 introduced several groundbreaking concepts that became standard in modern CNNs:

🎯 Gradient-Based Learning

First successful application of backpropagation to train convolutional networks end-to-end. This was revolutionary because previous approaches required hand-crafted features.

🔄 Weight Sharing

Same filter weights used across all spatial locations, dramatically reducing parameters while maintaining translation equivariance. This insight was crucial for scalability.

🏗️ Hierarchical Architecture

Progressive feature abstraction from simple edges to complex patterns. Each layer builds upon representations learned by previous layers.

🌟 Impact on Modern Deep Learning:

  • AlexNet (2012): Scaled up LeNet principles with ReLU and dropout
  • VGGNet: Extended depth with smaller 3×3 filters
  • ResNet: Added skip connections to LeNet-style architectures
  • Modern CNNs: Still follow the conv-pool-conv-pool pattern

📥 Input Layer

Input specifications for LeNet-5:

  • Size: 32×32 pixels
  • Channels: 1 (grayscale)
  • Data: Handwritten digits from MNIST
  • Preprocessing: Normalized to [-1, 1] range
  • Padding: Original MNIST (28×28) centered in 32×32 field
×

🎯 C3 - Second Convolutional Layer

C3 extracts more complex features:

  • Input: 14×14×6 from S2
  • Filters: 16 filters of size 5×5
  • Output: 10×10×16
  • Special Connection: Not fully connected to all input channels
  • Purpose: Combines features from multiple input maps
×

🏊 S4 - Second Subsampling Layer

S4 further reduces spatial dimensions:

  • Input: 10×10×16 from C3
  • Operation: 2×2 average pooling
  • Output: 5×5×16
  • Function: Similar to S2 with trainable weights
×

🎯 C5 - Third Convolutional Layer

C5 acts as both convolution and fully connected:

  • Input: 5×5×16 from S4
  • Filters: 120 filters of size 5×5
  • Output: 1×1×120 (effectively 120 neurons)
  • Nature: Functions like fully connected layer
×

🔗 F6 - Fully Connected Layer

F6 is the first traditional fully connected layer:

  • Input: 120 units from C5
  • Output: 84 units
  • Activation: Radial Basis Function (RBF)
  • Purpose: Feature compression and abstraction
×

📤 Output Layer

Output layer performs final classification:

  • Input: 84 units from F6
  • Output: 10 classes (digits 0-9)
  • Function: Radial Basis Function (RBF)
  • Decision: Nearest prototype in RBF space
