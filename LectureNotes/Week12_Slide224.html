Understanding Stratified K-Fold

Stratified K-Fold ensures each fold preserves the percentage of samples for each class, maintaining the same class distribution as the original dataset.

Implementation & Code

from sklearn.model_selection import StratifiedKFold
from sklearn.metrics import accuracy_score
import numpy as np

# Initialize Stratified K-Fold
skf = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)

# Perform cross-validation
scores = []
for train_idx, test_idx in skf.split(X, y):
    X_train, X_test = X[train_idx], X[test_idx]
    y_train, y_test = y[train_idx], y[test_idx]
    
    # Train model
    model.fit(X_train, y_train)
    
    # Evaluate
    y_pred = model.predict(X_test)
    scores.append(accuracy_score(y_test, y_pred))

print(f"CV Score: {np.mean(scores):.3f} ± {np.std(scores):.3f}")

Key Advantages

  • ✓ Class Balance Preservation
    Maintains original class distribution in each fold
  • ✓ Ideal for Imbalanced Datasets
    Ensures minority classes are represented in all folds
  • ✓ More Reliable Estimates
    Reduces variance in performance metrics
  • ✓ Consistent Evaluation
    Each fold is representative of the whole dataset
