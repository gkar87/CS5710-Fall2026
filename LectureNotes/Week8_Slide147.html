Backpropagation Algorithm

The Algorithm

  • 1 Forward Pass
    Compute predictions and cache intermediate values
  • 2 Compute Loss
    Calculate error between predictions and targets
  • 3 Backward Pass
    Compute gradients layer by layer, backwards
  • 4 Update Weights
    Apply gradients to update parameters

Click to visualize gradient flow through network

Key Equations

Output Layer Gradient:
dZ[L] = A[L] - Y
dW[L] = (1/m) · dZ[L] · A[L-1]ᵀ
db[L] = (1/m) · Σ(dZ[L])
Hidden Layer Gradient:
dA[l] = W[l+1]ᵀ · dZ[l+1]
dZ[l] = dA[l] ⊙ g'[l](Z[l])
dW[l] = (1/m) · dZ[l] · A[l-1]ᵀ
Weight Update Rule:
W[l] = W[l] - α · dW[l]
b[l] = b[l] - α · db[l]
(α = learning rate)
Chain Rule:
∂L/∂W[l] = ∂L/∂Z[l] · ∂Z[l]/∂W[l]
∂L/∂Z[l] = ∂L/∂A[l] · ∂A[l]/∂Z[l]
