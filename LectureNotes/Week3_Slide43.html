Logistic Regression

Core Concept

Logistic Regression is a linear model for binary classification that uses the logistic (sigmoid) function to model probabilities.
  • 📈 Linear Decision Boundary: Creates a linear separation in feature space
  • 🎯 Probability Output: Outputs probabilities between 0 and 1
  • 🔄 Sigmoid Transform: Maps linear combination to probability
  • 📊 Maximum Likelihood: Parameters learned via MLE
  • ⚡ Efficient: Fast training and prediction

Mathematical Formulation

Linear Combination:
z = w₀ + w₁x₁ + w₂x₂ + ... + wₙxₙ
Sigmoid Function:
σ(z) = 1 / (1 + e⁻ᶻ)
Prediction:
P(y=1|x) = σ(wᵀx + b)
Key Properties:
  • • σ(z) → 1 as z → ∞
  • • σ(z) → 0 as z → -∞
  • • σ(0) = 0.5
  • • Derivative: σ'(z) = σ(z)(1 - σ(z))
