SARSA Algorithm

What is SARSA?

SARSA (State-Action-Reward-State-Action) is an on-policy TD control algorithm that learns Q-values for the policy being followed.
S A R S A
Q(s,a) ← Q(s,a) + α[r + γQ(s',a') - Q(s,a)]
On-Policy Learning:
• Learns about policy being followed
• More conservative than Q-Learning
• Takes exploration into account

SARSA vs Q-Learning

Aspect SARSA Q-Learning
Policy Type On-Policy Off-Policy
Update Rule Q(s',a') max Q(s',a)
Behavior Conservative Optimistic
Exploration Considers it Ignores it
Cliff Walking Safe path Optimal path
Convergence To π policy To π* optimal

SARSA Algorithm

SARSA (on-policy TD control):

# Initialize Q-values arbitrarily
Initialize Q(s,a) for all s ∈ S, a ∈ A(s), arbitrarily
Q(terminal, ·) = 0

Repeat for each episode:
  Initialize S
  Choose A from S using policy derived from Q (e.g., ε-greedy)
  Repeat for each step of episode:
    Take action A, observe R, S'
    Choose A' from S' using policy derived from Q (e.g., ε-greedy)
    # SARSA update (on-policy)
    Q(S,A) ← Q(S,A) + α[R + γQ(S',A') - Q(S,A)]
    S ← S'; A ← A'
  until S is terminal
