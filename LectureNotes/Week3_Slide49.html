Softmax Regression

🎯 Softmax Function

Softmax regression directly models multi-class probabilities using the softmax function, ensuring all class probabilities sum to 1.
Softmax Function:

P(y = k | x) = exp(θₖᵀx) / Σⱼ exp(θⱼᵀx)

For K classes:
Σₖ P(y = k | x) = 1
Key Features:

• Multi-class generalization of logistic regression
• Outputs valid probability distribution
• Uses cross-entropy loss
• Single unified model

📊 Properties

  • 📈 Probability Distribution
    All outputs sum to 1, forming valid probabilities
  • 🔄 Differentiable
    Smooth function enabling gradient-based optimization
  • ⚖️ Winner-Take-All
    Amplifies differences between class scores
  • 🎯 Maximum Entropy
    Natural extension from binary to multi-class
Example: Raw scores [2, 1, 0] → Softmax [0.67, 0.24, 0.09]
Softmax vs One-vs-All Comparison
Softmax Regression
P(y=k|x) = exp(θₖᵀx) / Σⱼexp(θⱼᵀx)
✅ Advantages:
• Probabilities sum to 1
• Unified training process
• Natural multi-class extension
• Better calibrated probabilities

❌ Disadvantages:
• More complex optimization
• Requires all classes during training
One-vs-All
P(y=k|x) = σ(θₖᵀx)
(Independent classifiers)
✅ Advantages:
• Simple to implement
• Parallelizable training
• Works with any binary classifier
• Can handle new classes easily

❌ Disadvantages:
• Probabilities don't sum to 1
• Class imbalance issues
