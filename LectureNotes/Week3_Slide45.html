Maximum Likelihood Estimation

🎯 The MLE Concept

Maximum Likelihood Estimation (MLE) finds the parameters that make the observed data most likely. It answers: "What parameter values would make this data most probable?"
Likelihood Function:

L(θ) = ∏ᵢ₌₁ⁿ P(yᵢ|xᵢ, θ)

Log-Likelihood:
ℓ(θ) = Σᵢ₌₁ⁿ log P(yᵢ|xᵢ, θ)
Key Intuition:

If we observe data points, the "best" model parameters are those that would have made this exact data most likely to occur.

📊 MLE Steps

  • Step 1: Write Likelihood
    Express P(data|parameters) for all observations
  • Step 2: Take Log
    Convert product to sum: log-likelihood
  • Step 3: Differentiate
    Find ∂ℓ/∂θ for each parameter
  • Step 4: Set to Zero
    Solve ∂ℓ/∂θ = 0 for optimal θ
  • Step 5: Verify Maximum
    Check second derivative < 0
Why Log? Taking logarithm converts products to sums, making differentiation much easier while preserving the maximum.
MLE for Logistic Regression
Bernoulli Likelihood
P(y|x,θ) = σ(θᵀx)ʸ × [1-σ(θᵀx)]¹⁻ʸ
• y = 1: P = σ(θᵀx)
• y = 0: P = 1 - σ(θᵀx)
• Compact form handles both cases
Log-Likelihood
ℓ(θ) = Σᵢ[yᵢlog(σᵢ) + (1-yᵢ)log(1-σᵢ)]

where σᵢ = σ(θᵀxᵢ)
• Sum over all training examples
• No closed-form solution
• Requires iterative optimization
