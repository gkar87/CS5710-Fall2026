Vanilla RNN Architecture

RNN Structure

    Time Step:     t-1        t         t+1
                     ↓         ↓          ↓
    Input:         x[t-1]    x[t]      x[t+1]
                     ↓         ↓          ↓
    Hidden:    h[t-1] → → h[t] → → h[t+1]
                     ↓         ↓          ↓
    Output:        y[t-1]    y[t]      y[t+1]
  • Input Layer (x)
    Receives sequential data at each time step
  • Hidden State (h)
    Maintains memory from previous time steps
  • Output Layer (y)
    Produces predictions at each time step
  • Recurrent Connection
    Feeds hidden state back to itself

Mathematical Formulation

Hidden State Update:

h[t] = tanh(Whh·h[t-1] + Wxh·x[t] + bh)
Output Calculation:

y[t] = Why·h[t] + by
Weight Matrices:

• Wxh: Input to hidden weights
• Whh: Hidden to hidden weights (recurrent)
• Why: Hidden to output weights
• bh, by: Bias vectors
Key Properties:

• Parameter Sharing: Same weights across all time steps
• Temporal Processing: Processes one element at a time
• Memory: Hidden state carries information forward

Simple RNN Implementation

import numpy as np class VanillaRNN: def __init__(self, input_size, hidden_size, output_size): # Initialize weights self.Wxh = np.random.randn(hidden_size, input_size) * 0.01 self.Whh = np.random.randn(hidden_size, hidden_size) * 0.01 self.Why = np.random.randn(output_size, hidden_size) * 0.01 self.bh = np.zeros((hidden_size, 1)) self.by = np.zeros((output_size, 1)) def forward(self, inputs): h = np.zeros((self.Whh.shape[0], 1)) # Initial hidden state outputs = [] for x in inputs: # Update hidden state h = np.tanh(np.dot(self.Wxh, x) + np.dot(self.Whh, h) + self.bh) # Compute output y = np.dot(self.Why, h) + self.by outputs.append(y) return outputs, h
Advantages: Simple architecture, fewer parameters than fully connected networks

Limitations: Difficulty learning long-term dependencies, vanishing/exploding gradients
