Gini Impurity

🎯 Gini Impurity

Gini Impurity measures how often a randomly chosen element would be incorrectly labeled if randomly labeled according to the distribution.
Gini(S) = 1 - Σ p(c)²
Where:
  • • Gini(S) = Gini impurity of set S
  • • p(c) = Proportion of class c
  • • Range: 0 (pure) to 0.5 (binary)
Key Properties:
• Gini = 0: Pure node
• Gini = 0.5: Maximum impurity (binary)
• Used in CART algorithm
• Computationally efficient

📊 Gini vs Entropy

Aspect Gini Entropy
Formula 1 - Σp² -Σp×log(p)
Range 0 to 0.5 0 to 1
Computation Faster Slower (log)
Algorithm CART ID3, C4.5
Preference Isolates classes Balanced splits
When to use:
• Gini: Default choice, faster
• Entropy: When you need information gain

Step-by-Step Calculation:

Example: 80 Yes, 20 No
p(Yes) = 0.8, p(No) = 0.2
Gini = 1 - (0.8² + 0.2²)
Gini = 1 - (0.64 + 0.04) = 0.32
Interpretation:
Lower Gini = More pure
0.32 suggests moderate impurity
Good candidate for further splitting
