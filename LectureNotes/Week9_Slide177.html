🎭 Image Segmentation

Segmentation Types

🧩 Semantic Segmentation
Classifies every pixel into predefined categories (e.g., car, road, tree)
🔍 Instance Segmentation
Identifies and separates individual object instances within the same class
🌐 Panoptic Segmentation
Combines semantic and instance segmentation for comprehensive scene understanding

Key Architectures

🏗️ U-Net
Encoder-decoder with skip connections, excels in medical imaging
🔗 FCN (Fully Convolutional)
Replaces dense layers with convolutions for pixel-wise predictions
🎯 DeepLab
Uses atrous convolutions and pyramid pooling for multi-scale features
