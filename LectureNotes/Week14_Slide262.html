Bias in Machine Learning

Types of Bias

ML Bias occurs when algorithms produce systematically prejudiced results due to erroneous assumptions in the learning process.
  • 📜 Historical Bias
    Bias that already exists in the world and is reflected in data
  • 👥 Representation Bias
    Failure to sample from population equally
  • 📏 Measurement Bias
    Features and labels are proxies for desired quantities
  • 🔄 Aggregation Bias
    One-size-fits-all models fail for diverse groups
  • ✓ Evaluation Bias
    Benchmark data doesn't represent actual use population

Sources & Impact

  • 💾 Data Collection
    Sampling methods, selection criteria, historical records
  • 🏷️ Labeling Process
    Annotator bias, subjective judgments, inconsistent standards
  • 🎯 Feature Selection
    Proxy variables, correlated attributes, missing factors
  • ⚙️ Algorithm Design
    Optimization objectives, regularization, architecture choices
  • 🔁 Feedback Loops
    Self-reinforcing predictions, confirmation bias
