Policy Functions

What is a Policy?

A policy is a strategy that defines the agent's behavior by specifying which action to take in each state.
π(a|s)
Probability of taking action a in state s

Types of Policies

🎯 Deterministic Policy
Always selects the same action for a given state
🎲 Stochastic Policy
Selects actions probabilistically based on a distribution
🧠 Parameterized Policy
Uses parameters (like neural network weights) to define behavior
Policy Examples in Action
♟️
Chess Strategy
Map board positions to best moves
🚗
Driving Policy
Traffic situations → driving actions
🤖
Robot Control

💰
