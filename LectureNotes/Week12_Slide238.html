Statistical Testing

Two-Sample t-test
Compares means between two groups for continuous metrics like accuracy, latency, or revenue.
Use when: Comparing continuous metrics between control and treatment
Chi-Square Test
Tests independence for categorical outcomes like click-through rates or conversion rates.
Use when: Comparing proportions or categorical outcomes
Mann-Whitney U Test
Non-parametric alternative when data doesn't meet normality assumptions.
Use when: Data is skewed or doesn't follow normal distribution
Bootstrap Test
Resampling method that doesn't require distributional assumptions.
Use when: Complex metrics or when unsure about data distribution
Statistical Significance:
• p-value < 0.05: Statistically significant
• Effect size: Practical significance
• Confidence intervals: Uncertainty quantification
• Power analysis: Avoid false negatives

A/B Testing Best Practices

🎲
Proper Randomization
Ensure truly random assignment to avoid selection bias. Use consistent user IDs for repeated interactions.
📊
Sample Size Planning
Calculate required sample size before starting. Consider effect size, statistical power, and significance level.
🕐
Sufficient Duration
Run experiments long enough to account for weekly patterns, seasonality, and user behavior cycles.
🎯
Primary Metric Focus
Choose one primary metric to avoid multiple testing problems. Monitor secondary metrics for side effects.
📈
Early Stopping Rules
Pre-define stopping criteria to avoid peeking bias. Use sequential testing methods if early stopping is needed.
🔒
Guardrail Metrics
Monitor key business metrics to ensure the experiment doesn't harm critical outcomes.
