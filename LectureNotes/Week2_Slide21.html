Linear Regression Introduction

What is Linear Regression?

Linear Regression is a supervised learning algorithm that models the relationship between a dependent variable and one or more independent variables by fitting a linear equation to observed data.
y = β₀ + β₁x + ε
  • 📈 Continuous Output
    Predicts numerical values rather than categories
  • 📐 Linear Relationship
    Assumes a straight-line relationship between variables
  • 🎯 Prediction & Inference
    Used for both making predictions and understanding relationships
  • ⚡ Simple & Fast
    Computationally efficient and easy to interpret

Core Components

Key Terminology:
  • x: Independent Variable
    Input features (predictors, explanatory variables)
  • y: Dependent Variable
    Output/target we want to predict
  • β₀: Intercept
    Y-value when x = 0 (baseline)
  • β₁: Slope
    Change in y per unit change in x
  • ε: Error Term
    Random noise/unexplained variation
