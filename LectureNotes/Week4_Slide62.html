Information Gain and Entropy

📊 Entropy

Entropy measures the impurity or uncertainty in a dataset. High entropy means high disorder, low entropy means high purity.
H(S) = -Σ p(c) × log₂(p(c))
Where:
  • • H(S) = Entropy of set S
  • • p(c) = Proportion of class c
  • • log₂ = Logarithm base 2
Key Points:
• Entropy = 0: Pure node (all same class)
• Entropy = 1: Maximum disorder (50/50 split)
• Used in ID3, C4.5 algorithms

📈 Information Gain

Information Gain measures how much a feature reduces entropy when used to split the data.
IG(S,A) = H(S) - Σ |Sv|/|S| × H(Sv)
Where:
  • • IG(S,A) = Information gain
  • • H(S) = Original entropy
  • • Sv = Subset for value v
  • • |Sv|/|S| = Weighted average
Decision Rule:
Choose the feature with the highest information gain for splitting!

Example Calculation:

Dataset: 9 Yes, 5 No (Play Tennis?)
Step 1: p(Yes) = 9/14, p(No) = 5/14
Step 2: H(S) = -(9/14)×log₂(9/14) - (5/14)×log₂(5/14)
Result: H(S) = 0.940 bits
