Generator and Discriminator Networks

🎨

Generator (G)

Architecture Layers:
Input: Random Noise (z ~ N(0,1))
Dense Layer (128 → 256 units)
Batch Normalization + ReLU
Dense/Deconv Layers (256 → 512)
Upsampling Layers
Output: Generated Sample
🎯 Objective: Fool the discriminator by generating realistic samples
Loss: minG log(1 - D(G(z)))
🔍

Discriminator (D)

Architecture Layers:
Input: Real/Fake Sample
Conv/Dense Layer (input → 512)
LeakyReLU + Dropout
Conv/Dense Layers (512 → 256)
Feature Extraction Layers
Output: Probability (sigmoid)
🎯 Objective: Correctly classify real vs generated samples
Loss: maxD [log D(x) + log(1 - D(G(z)))]

⚔️ Training Dynamics

🔄
Alternating Training
Train D for k steps, then G for 1 step
⚖️
Balance Point
Neither network dominates the other
🎯
Nash Equilibrium
D(x) = D(G(z)) = 0.5
💪
Capacity Balance
Match network complexities
📈
Gradient Flow
Avoid vanishing/exploding gradients
🔧
Training Stability
Use tricks like spectral norm
